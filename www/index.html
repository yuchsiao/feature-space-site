Predictive Nonlinear Modeling: In scientific research, building a model for physical systems is a critical step toward understanding the phenomena and the future behaviors. My research aims to provide an automatic black-box approach of model identification, which can be applied not only to physical devices, but also to general nonlinear dynamical systems for time-series predictive modeling. The contributions of this work are: i) the learning algorithm generates nonlinear dynamical models through convex optimization, ii) the generated models are guaranteed to be numerical stable and energy conservative (passive), and iii) can be interconnected to form large-scale dynamical systems. I have successfully applied my algorithms for modeling nonlinear circuits, human arterial networks, and finance time-series stock price prediction.

CAPLET is an open-source capacitance extraction toolkit that computes field-solver accurate solutions using our developed ultra parallel-efficient algorithm, which is built on top of instantiable basis functions for the boundary element method.

Our freely released software covers the complete extraction flow from GDS2 layout files to capacitance matrices. This is, to the best of our knowledge, the first open-source extraction tool that incorporates geometry algorithms, basis function generation, 3D layout visualization, and parallelized field solver extraction altogether. CAPLET provides both intuitive GUI and command line interfaces. The underlying algorithm is specialized for VLSI interconnects in Manhattan geometries inside a uniform dielectric material.

Website. GitHub. SourceForge.

Nearby: a map-based app that helps you find popular attractions nearby you, measured in time.

Many apps help us book, reserve, and plan which cities we would like to visit, but very few effectively guide us to walk around the cities after landed. Nearby fetches tourism data from the Internet, calibrates the locations, and visualize the calculated travel times so that travelers can easily enjoy a spontaneous trip.

The prototyped idea is showcased HERE.

Eggplot: A C++ library that allows you to plot just as easily as in MATLAB.

It has never been easy to plot in pure C++. Many tools and extra packages have to be at hand before a simple sinusoidal wave can be shown on screen. Researchers typically need to work back and forth between tools, dump data and read them properly, and then realize one more iteration is needed.

Eggplot makes life easier. Simply use a MATLAB-like syntax to plot. Let ubiquitous gnuplot handle the rest.

Website. GitHub.
